# Java Code Lab
![](https://img.shields.io/badge/maven-3.5.x-green.svg) ![](https://img.shields.io/badge/junit-4.11-green.svg)

Java 实现的数据结构/算法以及计算机科学相关的代码和完整的 Junit 测试用例。

## 运行

```shell
项目根目录下

# 测试所有
> mvn test 

# 测试指定代码
> mvn test -Dtest=完整类名

例如
# 测试排序算法类
> mvn test -Dtest=org.razertory.javacodelab.sort.SortTest
```
